# SOC Lab - Security Operations Center Training

This repository contains documentation and analysis of security incidents handled in a SOC environment. Each incident is documented with the tools used, findings, and actions taken to resolve the situation.

## Overview

A Security Operations Center (SOC) is responsible for monitoring, detecting, and responding to security incidents. This lab provides hands-on experience with incident investigation using real-world tools and scenarios.

## Tools Used in These Incidents

- **Firewall**: Network access control and threat blocking
- **Splunk**: Log search and analysis
- **VirusTotal**: URL and file reputation checking
- **Alert Management System**: Incident detection and tracking
